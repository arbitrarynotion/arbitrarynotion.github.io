Space Sim

Space Sim is a space-themed Unity game where AI bots work to simulate a dynamic economy in a procedurally generated world.

Initially created as a university project, I've since entirely overhauled and expanded the project on all levels to include cleaner code, in-editor real-time world generation customization, more efficient algorithms, more complete UI, and far better visuals.

ACE UI Framework

Ace UI Framework is a Unity plugin that makes creating custom Inspectors in Unity easier while providing a suite of stylization and data visualization features.

This project had the combined goals of improving the game designer's experience with custom script's Inspectors and giving me an excuse to deep dive into Unity's editor code.

Procedural Generation Research

Procedural Generation Research was a project designed to explore the benefits of utilizing procedural generation to efficiently produce meaningful training scenarios for medical diagnosis training.

This was a team research project where I contributed in discovering and defining our application's research objective. I then designed and implemented the procedural generation algorithm which generated dynamic diagnosis subjects.

Experience Summary

Have a specific role in mind? Check out the list below to see a breakdown of many experience.
Of course, this list is just an overview so if you don't see what you're looking be sure to get in touch with me, and I'll be happy to answer your questions.

  • UI development, including animations, interactive elements, and UI attached to moving objects within the game world.
  • Building chained animations with DoTween to enhance game feel.
  • Prototyping with ProBuilder and assigning materials and colors at runtime.
  • Using Particle Systems to create mining effects, collection handled by drawing particles to a collider with a force field and counting impacts.
  • Navigation using Unity's NavMesh system, baked at runtime after procedurally generated resources are instantiated to incorporate their static positions without the performance hit of NavMesh Obstacles.
  • Camera layering to create the illusion of a background of nebulas and planets.
  • Built-in Renderer Post Processing.
  • Addressing performance issues with Unity's Profiler both in-editor and in-game.
  • Developed specialized logging tools to streamline debugging with toggleable, class-specific and color coded console output.
  • Developing procedural generation algorithms that must perform quickly while minimizing performance impact to show results in real-time.
  • Developing in-game searching algorithms for locating resources without impacting performance.
