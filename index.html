Featured Work

Space Sim

Space Sim is a space-themed Unity game economy simulation where AI bots work to develop a dynamic economy in a procedurally generated world.

Initially created as a university project, I've since entirely overhauled and expanded the project on all levels to include cleaner code, in-editor real-time world generation customization, more efficient algorithms, more complete UI, and better visuals.

Project Page

ACE UI Toolkit

Ace UI Toolkit is a Unity plugin that makes creating custom Inspectors in Unity easier while providing a suite of stylization and data visualization features.

This project has the combined goals of improving the game designer's experience with custom script's Inspectors and giving me an excuse to deep dive into Unity's editor code.

Project Page

Procedural Generation Research

Procedural Generation Research was a project designed to explore the benefits of utilizing procedural generation to efficiently produce meaningful training scenarios for medical diagnosis training.

This was a team research project where I contributed in discovering and defining our application's research objective. I then designed and implemented the procedural generation algorithm which generated dynamic diagnosis subjects.
